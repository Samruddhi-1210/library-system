# library-system
Project Description

The Library System is a web-based application developed to simplify the management of library operations. It helps librarians manage books, users, and book transactions efficiently while reducing manual paperwork.
The system provides a user-friendly interface for managing books, issuing and returning books, and maintaining library records.

 Features
 Admin Login
 User Registration & Login
 Add New Books
 Edit Book Details
 Delete Books
 Search Books
 View Available Books
 Issue Books
 Return Books
 Manage Categories
 Dashboard with Statistics
